## Switch Statement
C# includes another decision making statement called switch. The switch statement executes the code block depending upon the resulted value of an expression.
### Syntax
```javascript
switch(expression)
{
    case value1:
        // code block
        break;
    case value2:
        // code block
        break;
    case valueN:
        // code block
        break;
    default: 
        // code block
        break;
}
```
